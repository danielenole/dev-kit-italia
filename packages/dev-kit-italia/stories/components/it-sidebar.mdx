## Cosa fa

Utilizza il componente Sidebar per navigare tra le pagine che appartengono a una sezione di un sito. A seconda dell'organizzazione dei contenuti, la barra di navigazione può presentare liste di link diretti o annidati sotto una voce che li raccoglie.

[Approfondisci quando e come usare il componente Sidebar](https://designers.italia.it/design-system/componenti/sidebar/).

## Accessibilità

- Indica sempre **visivamente** la pagina corrente utilizzando la classe `.active` sul link corrispondente
- Indica sempre **alle tecnologie assistive** la pagina corrente utilizzando l'attributo `aria-current="page"` sul link corrispondente
- Usa l'attributo `aria-disabled="true"` per i link disabilitati per garantire il corretto supporto alle tecnologie assistive
- Per le liste collassabili, il componente `it-collapse` gestisce automaticamente gli attributi ARIA necessari (`aria-expanded`, `aria-controls`, `role`) per garantire l'accessibilità



## Sidebar con icone

La Sidebar può contenere icone nella sua lista di link, utilizzando il componente `<it-icon>` con classe `.left` all'interno dei link con classe `.left-icon`.

<Canvas of={SidebarStories.SidebarConIcone} />

## Sidebar con linea a destra

Per creare una sidebar con linea separatrice a destra è sufficiente aggiungere la classe `.it-line-right-side` al wrapper più esterno.

<Canvas of={SidebarStories.SidebarLineaDestra} />

## Sidebar con linea a sinistra

Per creare una sidebar con linea separatrice a sinistra è sufficiente aggiungere la classe `.it-line-left-side` al wrapper più esterno.

<Canvas of={SidebarStories.SidebarLineaSinistra} />

## Sidebar annidata

La Sidebar può contenere una lista di link primaria annidata utilizzando la classe `.link-sublist` per le sotto-liste espanse.

Per rendere collassabili le sottosezioni della sidebar, utilizza il componente `it-collapse` integrato nelle liste di navigazione.

Il componente `it-collapse` permette di espandere e comprimere le sottosezioni in modo interattivo, migliorando l'esperienza di navigazione su sidebar con molte voci.

Consulta la documentazione del [componente Collapse](/docs/componenti-collapse--documentazione) per maggiori dettagli sull'uso di `it-collapse`.
