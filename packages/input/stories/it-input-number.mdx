# Input Numerico

<description>Campo di inserimento di un valore numerico, con pulsanti per incrementarlo o decrementarlo</description>

## Cosa fa
Il componente input numerico, o numberinput, consente all’utente di inserire una numero specifica, sia digitando manualmente il valore, sia incrementando/decrementando il valore con gli appositi pulsanti.
È utile in situazioni in cui si voglio inserire valori relativi a quantità, codici numerici, intervalli di valori. Aiuta a prevenire errori, e rende più facile inserire i dati per l'utente.

[Approfondisci come e quando usare il componente Input](https://designers.italia.it/design-system/componenti/number-input/)

## Indicazioni generali

L’input di tipo numerico è una variante del componente `<it-input>` con l'attributo `type=` impostato su `number` (`type="number"`).

Pertanto, per quanto riguarda:

- la **validazione** e la **gestione degli errori**
- la **gestione degli eventi**
- i **metodi e le proprietà** accessibili tramite JavaScript
- il **supporto all’internazionalizzazione (i18n)**

è necessario fare riferimento alla [documentazione di it-input](?path=/docs/componenti-form-input--documentazione).

## Accessibilità

Tutti gli attributi `it-aria-*` passati a `<it-input>` vengono applicati all'input generato.
