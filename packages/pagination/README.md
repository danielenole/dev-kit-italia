# Pagination

Componente per la navigazione tra pagine di contenuto.

## Installazione

```bash
npm install @italia/pagination
```

## Uso

```html
<it-pagination value="3">
  <it-pagination-item page="1">
    <a href="?page=1">1</a>
  </it-pagination-item>
  <it-pagination-item page="2">
    <a href="?page=2">2</a>
  </it-pagination-item>
  <it-pagination-item page="3">
    <a href="?page=3">3</a>
  </it-pagination-item>
</it-pagination>
```
